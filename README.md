## Qui je suis

Après plusieurs années dans l'ingénieurie et l'organisation de grands événements — dont les **Jeux Olympiques Paris 2024** — j'ai opéré une reconversion complète vers l'analyse data et vidéo dans le football.

Ce n'est pas un virage à 180°. C'est la même logique : **comprendre des systèmes complexes, extraire ce qui compte, et le rendre actionnable**. Sauf qu'aujourd'hui, le terrain de jeu c'est un rectangle vert et les 22 joueurs qui s'y déplacent.

Certifiée en **Sports Analytics** (Université du Michigan) et **Analyse Vidéo & Data dans le Sport** (Université de Lorraine), je maîtrise à la fois le code (Python, StatsBomb, SkillCorner) et les outils vidéo professionnels (Dartfish, Once Sport, Nacsport).

> Ce qui me distingue : je fais les deux. Data **et** vidéo. Chiffres **et** images. C'est rare — et c'est délibéré.

## 🎯 Projet Phare — ScoutPro 

> **La plateforme de scouting football la plus complète de mon portfolio.**

Interface de recrutement data-driven inspirée des cartes FIFA, enrichie de vraies données StatsBomb sur **43 compétitions** et **6 343 joueurs**.

- Recherche multi-critères, fiche FIFA-like, matching tactique
- **Scout Report** : xG/90, passes progressives, pressing, percentiles vs groupe de poste
- Données réelles sur des centaines de matchs — Messi 481 buts / 560 matchs, Mbappé 62 buts / 76 matchs

**Stack :** Python · StatsBomb · soccerdata · Streamlit · Plotly
